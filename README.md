# VITAlbum

A simple homebrew file browser that is used for viewing various image formats on the PlayStation VITA.
<img src="https://i.imgur.com/KI6woYJ.jpg" alt="3DShell Screenshot" width="640" height="362"/>

# Supported Formats:
- BMP
- GIF
- ICO
- JPG/JPEG
- PCX
- PNG
- PGM/PPM
- PSD
- TGA
- TIFF
- WEBP
